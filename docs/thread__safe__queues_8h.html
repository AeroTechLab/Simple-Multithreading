Detailed Description

Abstractions for thread safe queue data structures.

Queues that can be safely (no data races) accessed by different concurrent threads

Enumeration Type Documentation

◆ TSQueueAccessMode

enum TSQueueAccessMode

Option to control behaviour of queue access on empty or full cases.

Enumerator
TSQUEUE_WAIT 

Wait for availability of data for reading or space for writing.

TSQUEUE_NOWAIT 

Automatically return when reading empty queue or overwrite when writing to full queue.
